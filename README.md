# 1. Set up Database MongoDB trước khi chạy ứng dụng.
### Bước 1: Tạo database với tên "Data_Object"
### Bước 2: tạo 3 collection lần lượt với tên "customers", "employees", "transactions"
### Bước 3: import các file CSV trong thư mục Sample_data. Cụ thể:
- Đối với collection "customers": bấm ADD DATA --> Import JSON or CSV file --> chuyển đến thư mục Sample_data cùng cấp với README.md trong dự án-->import file Members_Data.csv
**Lưu ý: Ở màn hình import điều chỉnh kiểu dữ liệu của 2 biến "JoinDate" và "LastPurchaseDate" thành dạng Date.**

- Đối với collection "employees": bấm ADD DATA --> Import JSON or CSV file --> chuyển đến thư mục Sample_data cùng cấp với README.md trong dự án-->import file Data_Object.employees.csv
**Lưu ý: Ở màn hình import điều chỉnh kiểu dữ liệu của biến "password" thành dạng string.**
- Đối với collection "transactions": bấm ADD DATA --> Import JSON or CSV file --> chuyển đến thư mục Sample_data cùng cấp với README.md trong dự án-->import file Retail_Transaction_Dataset.csv
**Lưu ý: Ở màn hình import điều chỉnh kiểu dữ liệu của biến "TransactionDate" thành dạng Date.**
